    1. Create src/hooks/useCall.ts per research Pattern 2:
       - Signature: `useCall(peerRef: RefObject<Peer | null>, streamRef: RefObject<MediaStream | null>, roomId: string)`
       - Internal refs: `callRef` (MediaConnection | null), `remoteStreamRef` (MediaStream | null)
       - `subscribeToCall(call)`:
         - `call.on('stream')` → set remoteStreamRef.current, setConnectionState('connected'), setWasConnected(true), playJoinSound()
         - `call.on('close')` → check wasConnected via useCallStore.getState().wasConnected: if true → setCallEnded(true) + playLeaveSound(); if false → setConnectionState('failed') (Pitfall 3). Always set remoteStreamRef.current = null
         - `call.on('error')` → console.error, setConnectionState('failed')
       - useEffect on [roomId]:
         - Guard: if (!peer || !stream) return
         - isJoiner = peer.id !== roomId
         - Joiner: peer.call(roomId, stream) → subscribeToCall
         - Creator: peer.on('call', (call) => { call.answer(stream); subscribeToCall(call) })
         - Cleanup: callRef.current?.close(), null refs
       - hangUp callback: close call, null refs, setConnectionState('disconnected'), setCallEnded(true)
       - Return: { remoteStreamRef, hangUp }
